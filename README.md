# catSNS

https://onigiri860.github.io/catSNS/

SNSを見ているときになんとなくおすすめの表示を更新してしまうことはないか？

しかもそういう更新をした時に限って興味がありそうな投稿が画面が更新される直前に見える。
しかし、その投稿をおすすめ欄をさかのぼっても全然出てこなくてあきらめてしまう。

ここに少し面白さを感じた。

そこで、更新前のSNSの投稿を探すゲームを作ろうと思う。

簡易的に作るならAIの投稿を無数に作成し、決められた一つの投稿（X）を更新後に探すゲーム。
乱数で何個下の投稿に追加するとかを決めてスクロールすればいい。
これは暇人のいい暇つぶしになると考える。

後はデザインの問題なのだがAIの投稿される文字を全部独自でデザインした新しい文字を用いようかなと考えている。
例えばヒエログリフのような文字。

ここにいい案があれば追加してほしい。

・似たようなフェイクを作成

・画面の一番上や下の触れてしまいそうなところに更新ボタンを配置

・広告のようなものを追加するリアルさ
